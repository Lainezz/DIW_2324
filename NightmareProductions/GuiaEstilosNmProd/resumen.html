PUBLICO OBJETIVO

El público objetivo del sitio web es eminentemente un público juvenil profesionalizado. Este público se dirigirá al sitio web para informarse sobre las diferentes producciones que la empresa Nightmare Productions ha realizado.

Al ser un público joven, los colores que se aportan pueden ser un poco más agresivos y más llamativos, ya que se espera que este tipo de público esté familiarizado con este tipo de colores.

Por otro lado, el tipo de público que se espera también tendría un perfil un tanto profesionalizado, ya que se piensa que la gente que visite el sitio web pertenecerá a un nicho bastante concreto, relacionado con el mundo cinematográfico.

TIPO DE WEB

El tipo de web será una web eminentemente informativa, donde se expondrán las diferentes películas realizadas por la productora, las diferentes visitas que la productora ofrece, y un apartado sobre nosotros, donde se expondrá la historia de la productora.

ELEMENTOS DE DISEÑO

Como elementos de diseño, se hará hincapié en algunos aspectos:

  • Colores relacionados con temática halloween, zombies, miedo, etc
  • Tipografía relacionada con la misma temática que se indica en el punto anterior
